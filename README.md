# ABP_UserSystem
基本实现角色权限管理，具有很高的学习价值。

# UI
UI主要使用Bootstrap以及相关的扩展，还有一部分自行封装的js。
系统目前能够适配IE8+的游览器，保证大部分的特效、功能使用。
推荐在Chrome上使用、调试本程序。
